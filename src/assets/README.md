# Recursos visuais

Esta pasta receberá os recursos utilizados pelo Flet e pelo empacotamento.

Arquivos planejados:

- `icon.ico`: ícone do executável do Windows;
- `logo.png`: logotipo exibido na interface;
- imagens usadas pela documentação e pelas telas.

Não coloque a imagem integral da norma no executável. A aplicação deve armazenar somente a
identificação da fonte e os parâmetros necessários às regras implementadas.
